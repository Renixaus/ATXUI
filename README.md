# ATX_UI

A feature-rich Roblox UI library for scripts — smooth micro-animations, glass effects, theming, and a full element suite.

> [!WARNING]
> ATX_UI is currently in Beta. Bugs and unstable features may occur.

## Features

- Window system with sidebar navigation, drag, resize, fullscreen
- 17+ elements: Button, Toggle, Slider, Dropdown, Input, Keybind, Colorpicker, Paragraph, Code, Label, Image, Divider, Group, HStack, VStack, Section, Space
- Smooth micro-animations — press feedback, focus glow, spring transitions (Back, Sine easings)
- Mac-style and Default topbar variants
- Dark / Light themes + custom theme API + gradient support
- ConfigManager (save/load JSON per-user), KeySystem (Platoboost + custom validator), Localization
- Notification, Dialog, Popup, Tooltip
- Acrylic/blur background, CustomEmptyPage, TabTitleAlign

## Installation

```luau
loadstring(game:HttpGet('https://raw.githubusercontent.com/Renixaus/ATXUI/refs/heads/main/dist/main.lua'))()
```
